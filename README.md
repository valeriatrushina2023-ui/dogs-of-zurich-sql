# Dogs of Zurich — SQL Analysis

## Project Overview
This project explores dog registration data in Zurich using SQL in Google BigQuery.
The goal is to analyse breed distribution, geographic patterns, and gender structure of the dog population.

## Dataset
Source: Kaggle — Dogs of Zurich
The dataset contains official dog registration records from Zurich.

## Tools Used
- Google BigQuery
- SQL
- GitHub

## Key Questions
1. How many unique districts are represented?
2. What is the distribution of dog breeds?
3. What is the gender distribution of dogs?

 ## Key Findings
- The dataset contains 12 unique districts.
- Further insights can be found in the SQL queries inside `analysis.sql`.

## Files
- `analysis.sql` — contains all SQL queries used in the analysis.
